GEOSQUARE INTERVIEW CHALLENGE

Grid Downscaling Service

Get interpolated value from GID :
{url}/grid/gid/{gid}
example :
{url}/grid/gid/J3N2M753E7F7E7
output :
{"gid":"J3N2M753E7F7E7","value":3.991540631360183}
Get interpolated value from Longitude Latitude :
{url}/grid/gid/{longitue}/{latitude}
example :
{url}/grid/coordinate/106.74420626/-6.22748512
output :
{"gid":"J3N2M753E7F7E7","value":3.991540631360183}
